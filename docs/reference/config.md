# config.json 設定リファレンス

`config.json` は、AriaEngine のユーザー設定を保存するJSONファイルです。エンジン起動時に自動読み込みされ、設定変更時に自動保存されます。ファイルが存在しない場合は、すべての設定がデフォルト値で動作します。

## 設定一覧

### テキスト表示

#### `GlobalTextSpeedMs`
- **型**: `int`
- **デフォルト値**: `30`
- **説明**: テキストの表示速度（タイプライター効果の待ち時間）をミリ秒で指定します。値が小さいほど高速に表示されます。
- **有効な値**: `0` 以上の整数。`0` で即時表示になります。

#### `DefaultTextSpeedMs`
- **型**: `int`
- **デフォルト値**: `30`
- **説明**: エンジンのデフォルトテキスト速度です。スクリプトで `textspeed` コマンドを使用して変更された場合、`GlobalTextSpeedMs` と同じ値に更新されます。
- **有効な値**: `0` 以上の整数

#### `TextMode`
- **型**: `string`
- **デフォルト値**: `"adv"`
- **説明**: テキスト表示モードを指定します。`adv` は画面下部にテキストボックスを表示するADVモード、`nvl` は画面全体にテキストを表示するNVLモードです。
- **有効な値**: `"adv"` または `"nvl"`

#### `SkipUnread`
- **型**: `bool`
- **デフォルト値**: `false`
- **説明**: 未読のテキストもスキップするかどうかを指定します。`true` の場合、既読・未読に関わらずスキップモードで全テキストを飛ばします。
- **有効な値**: `true` または `false`

#### `AutoModeWaitTimeMs`
- **型**: `int`
- **デフォルト値**: `2000`
- **説明**: オートモード時の、ページ送りまでの待ち時間をミリ秒で指定します。
- **有効な値**: `0` 以上の整数

### オーディオ

#### `BgmVolume`
- **型**: `int`
- **デフォルト値**: `100`
- **説明**: 背景音楽（BGM）の音量を指定します。
- **有効な値**: `0` ～ `100`

#### `SeVolume`
- **型**: `int`
- **デフォルト値**: `100`
- **説明**: 効果音（SE）の音量を指定します。
- **有効な値**: `0` ～ `100`

### 画面表示

#### `IsFullscreen`
- **型**: `bool`
- **デフォルト値**: `false`
- **説明**: フルスクリーンモードのON/OFFを指定します。ゲーム内で `F11` キーまたはスクリプトから切り替えると自動保存されます。
- **有効な値**: `true`（フルスクリーン）または `false`（ウィンドウ）

#### `WindowWidth`
- **型**: `int`
- **デフォルト値**: `1280`
- **説明**: ウィンドウモード時の画面幅をピクセルで指定します。フルスクリーンからウィンドウに戻る際に使用されます。
- **有効な値**: `1` 以上の整数

#### `WindowHeight`
- **型**: `int`
- **デフォルト値**: `720`
- **説明**: ウィンドウモード時の画面高さをピクセルで指定します。フルスクリーンからウィンドウに戻る際に使用されます。
- **有効な値**: `1` 以上の整数

## 設定例

```json
{
  "GlobalTextSpeedMs": 30,
  "DefaultTextSpeedMs": 30,
  "BgmVolume": 100,
  "SeVolume": 100,
  "IsFullscreen": false,
  "TextMode": "adv",
  "SkipUnread": false,
  "AutoModeWaitTimeMs": 2000,
  "WindowWidth": 1280,
  "WindowHeight": 720
}
```

## 備考

- 設定ファイルは `config.json`（エンジン実行ファイルと同じディレクトリ）に保存されます。
- 不正なJSON形式の場合、エラーメッセージを表示した上でデフォルト値で起動します。
- 一部の設定（音量、テキスト速度など）はゲーム内メニューから変更可能です。
