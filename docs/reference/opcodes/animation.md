## イージング設定

### `ease`

以降のアニメーションコマンド（`amsp`, `afade`, `ascale`）で使用するイージング関数を設定します。

```aria
ease type
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `type` | 文字列 | `"linear"` / `"easein"` / `"easeout"` / `"easeinout"` |

**例:**
```aria
ease "easeout"
amsp 1, 500, 200, 500       ; easeoutで移動
```

**注意:** デフォルトは`"linear"`です。`ease`の設定はグローバルで、次に`ease`を呼ぶまで維持されます。

**イージングの種類:**

| 名前 | 数式 | 説明 |
|------|------|------|
| `linear` | 線形 | 一定速度で変化 |
| `easein` | 二次加速 | ゆっくり始まって加速 |
| `easeout` | 二次減速 | 速く始まって減速 |
| `easeinout` | 二次加速減速 | ゆっくり始まり、中盤で加速し、最後に減速 |

## スプライトアニメーション

### `amsp`

スプライトをアニメーション付きで移動します。X座標とY座標を同時に補間します。

```aria
amsp id, toX, toY, duration
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `toX` | 整数 | 移動先X座標 |
| `toY` | 整数 | 移動先Y座標 |
| `duration` | 整数 | アニメーション時間（ミリ秒） |

**例:**
```aria
ease "easeout"
amsp 1, 500, 200, 500       ; 500msかけて(500, 200)へ移動
```

**注意:** 現在の`ease`設定が適用されます。スプライトが存在しない場合は何も起こりません。

---

### `afade`

スプライトをアニメーション付きでフェードします。不透明度を補間します。

```aria
afade id, toAlpha, duration
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `toAlpha` | 整数 | 目標不透明度（0〜255）。0=完全透明、255=完全不透明 |
| `duration` | 整数 | アニメーション時間（ミリ秒） |

**例:**
```aria
afade 1, 0, 1000            ; 1000msかけて透明に
afade 1, 255, 500           ; 500msかけて完全表示
```

**注意:** 内部では`toAlpha`を`255.0f`で除算し、0.0〜1.0の不透明度として扱います。

---

### `ascale`

スプライトをアニメーション付きで拡大・縮小します。X方向とY方向を同時に補間します。

```aria
ascale id, toScaleX, toScaleY, duration
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `toScaleX` | 数値 | 目標X拡大率 |
| `toScaleY` | 数値 | 目標Y拡大率 |
| `duration` | 整数 | アニメーション時間（ミリ秒） |

**例:**
```aria
ascale 1, 1.5, 1.5, 300     ; 300msかけて1.5倍に拡大
ascale 1, 1.0, 1.0, 200     ; 200msかけて元のサイズに戻す
```

**注意:** 現在の`ease`設定が適用されます。

---

### `acolor`

スプライトの色を即座に変更します。

```aria
acolor id, color
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `color` | 文字列 | 新しい色（`#rrggbb`） |

**例:**
```aria
acolor 10, "#ff0000"        ; テキストを即座に赤に
```

**注意:** `acolor`はアニメーション**ではなく**即座に色を変更します。名前に`a`がついていますが、トゥイーン対象外です。`TweenProperty`に`Color`が存在しないため、現在の実装では補間できません。

## アニメーション制御

### `await`

アニメーションの完了を待ちます。

```aria
await
```

**例:**
```aria
amsp 1, 500, 200, 500
await                       ; 移動が終わるまで待機
text "移動完了しました"
```

**注意:** `await`を呼ぶと、VMは`WaitingForAnimation`状態になり、すべてのトゥイーンが終了すると自動的に復帰します。タイプライター効果が完了していない場合は、そちらも待ちます。

## 画面フェード

### `fade_in`

画面をフェードインします。黒画面からゲーム画面へ徐々に明るくなります。

```aria
fade_in [duration]
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `duration` | 整数 | フェード時間（ミリ秒）。省略時は1000ms |

**例:**
```aria
fade_in         ; 1000msでフェードイン
fade_in 500     ; 500msでフェードイン
```

**注意:** フェード中はVMが`FadingIn`状態になり、完了すると自動的に`Running`に戻ります。

---

### `fade_out`

画面をフェードアウトします。ゲーム画面から黒画面へ徐々に暗くなります。

```aria
fade_out [duration]
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `duration` | 整数 | フェード時間（ミリ秒）。省略時は1000ms |

**例:**
```aria
fade_out        ; 1000msでフェードアウト
fade_out 2000   ; 2秒かけてフェードアウト
```

**注意:** フェード中はVMが`FadingOut`状態になり、完了すると自動的に`Running`に戻ります。

## 画面揺れ

### `quake`

画面を揺らします。すべてのスプライトが一緒に揺れます。

```aria
quake [amplitude] [time]
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `amplitude` | 整数 | 揺れの振幅（ピクセル）。省略時は5 |
| `time` | 整数 | 揺れ時間（ミリ秒）。省略時は500 |

**例:**
```aria
quake           ; デフォルト（振幅5、500ms）
quake 10, 1000  ; 振幅10、1秒間揺れる
```

**注意:** 揺れは描画オフセットに適用され、すべてのスプライトに一括で影響します。`quakex`というエイリアスも同じ動作です。
