## スプライト作成

### `lsp`

画像スプライトを読み込みます。

```aria
lsp id, "path", x, y
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID。0は背景専用、1〜をキャラクターやUIに使用 |
| `path` | 文字列 | 画像ファイルパス（`assets/bg/`や`assets/ch/`からの相対パス） |
| `x` | 整数 | X座標（左上基点） |
| `y` | 整数 | Y座標（左上基点） |

**例:**
```aria
lsp 1, "ch/mio_normal.png", 400, 100
```

**注意:** スプライトIDは重複すると上書きされます。スコープ内で作成したスプライトはスコープ終了時に自動削除されます。

---

### `lsp_text`

テキストスプライトを作成します。

```aria
lsp_text id, "text", x, y
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `text` | 文字列 | 表示するテキスト |
| `x` | 整数 | X座標 |
| `y` | 整数 | Y座標 |

**例:**
```aria
lsp_text 10, "HP: 100", 20, 20
sp_color 10, "#ff0000"
```

**注意:** 作成時のフォントサイズと色は、現在の`fontsize`/`textcolor`設定を継承します。後から`sp_fontsize`/`sp_color`で変更できます。

---

### `lsp_rect`

矩形スプライトを作成します。

```aria
lsp_rect id, x, y, width, height
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `x` | 整数 | X座標 |
| `y` | 整数 | Y座標 |
| `width` | 整数 | 幅（ピクセル） |
| `height` | 整数 | 高さ（ピクセル） |

**例:**
```aria
lsp_rect 100, 300, 400, 200, 80
sp_fill 100, "#333333", 200
sp_round 100, 12
```

**注意:** 作成直後は黒（`#000000`）の不透明な矩形です。`sp_fill`で色と透明度を設定してください。

## 表示・削除

### `vsp`

スプライトの表示・非表示を切り替えます。

```aria
vsp id, on/off
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `on/off` | 文字列/整数 | `"on"`/`1`で表示、`"off"`/`0`で非表示 |

**例:**
```aria
vsp 1, off      ; 非表示
vsp 1, on       ; 表示
vsp 1, 0        ; 非表示（数値でも可）
```

---

### `csp`

スプライトを削除します。

```aria
csp id
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID。`-1`を指定すると**すべてのスプライト**を削除 |

**例:**
```aria
csp 1           ; ID 1のスプライトを削除
csp -1          ; 全スプライト削除
```

**注意:** `csp -1`はボタン情報、フォーカス状態、スプライト寿命管理もすべてクリアします。

## 移動

### `msp`

スプライトを絶対座標に移動します。

```aria
msp id, x, y
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `x` | 整数 | 移動先X座標 |
| `y` | 整数 | 移動先Y座標 |

**例:**
```aria
msp 1, 500, 200     ; スプライト1を(500, 200)へ
```

---

### `msp_rel`

スプライトを相対座標で移動します。

```aria
msp_rel id, dx, dy
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `dx` | 整数 | X方向の移動量 |
| `dy` | 整数 | Y方向の移動量 |

**例:**
```aria
msp_rel 1, 10, 0    ; 右へ10ピクセル移動
msp_rel 1, 0, -5    ; 上へ5ピクセル移動
```

## 基本プロパティ

### `sp_z`

スプライトのZ順（重ね順）を設定します。

```aria
sp_z id, z
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `z` | 整数 | Z値。大きい値ほど手前に表示 |

**例:**
```aria
sp_z 1, 10      ; スプライト1をZ=10に
sp_z 2, 20      ; スプライト2が手前に表示
```

**注意:** 背景（ID 0）は通常Z=0です。UI要素には大きなZ値を割り当ててください。

---

### `sp_alpha`

スプライトの不透明度を設定します。

```aria
sp_alpha id, alpha
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `alpha` | 整数 | 不透明度（0〜255）。0で完全透明、255で完全不透明 |

**例:**
```aria
sp_alpha 1, 128     ; 半透明
sp_alpha 1, 255     ; 完全不透明
```

---

### `sp_scale`

スプライトの拡大率を設定します。

```aria
sp_scale id, scaleX, scaleY
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `scaleX` | 数値 | X方向の拡大率。1.0が等倍 |
| `scaleY` | 数値 | Y方向の拡大率。1.0が等倍 |

**例:**
```aria
sp_scale 1, 1.5, 1.5    ; 1.5倍に拡大
sp_scale 1, 2.0, 0.5    ; X方向に伸ばす
```

---

### `sp_fontsize`

テキストスプライトのフォントサイズを変更します。

```aria
sp_fontsize id, size
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `size` | 整数 | フォントサイズ（ピクセル） |

**例:**
```aria
sp_fontsize 10, 32      ; テキストを32pxに
```

---

### `sp_color`

テキストスプライトの色を変更します。

```aria
sp_color id, color
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `color` | 文字列 | 色コード（`#rrggbb`または`#rgb`） |

**例:**
```aria
sp_color 10, "#ff0000"      ; 赤色
sp_color 10, "#0f0"         ; 緑色（短縮形）
```

---

### `sp_fill`

矩形スプライトの塗りつぶし色と透明度を設定します。

```aria
sp_fill id, color, alpha
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `color` | 文字列 | 塗りつぶし色（`#rrggbb`） |
| `alpha` | 整数 | 不透明度（0〜255） |

**例:**
```aria
sp_fill 100, "#ff0000", 128     ; 赤の半透明
```

---

### `sp_rotation`

スプライトの回転角度を設定します。

```aria
sp_rotation id, angle
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `angle` | 整数 | 回転角度（度）。時計回り |

**例:**
```aria
sp_rotation 1, 45       ; 45度回転
sp_rotation 1, 0        ; 回転をリセット
```

## 装飾

### `sp_round`

矩形スプライトの角丸半径を設定します。

```aria
sp_round id, radius
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `radius` | 整数 | 角丸の半径（ピクセル）。0で直角 |

**例:**
```aria
sp_round 100, 12        ; 12pxの角丸
```

---

### `sp_border`

矩形スプライトの枠線を設定します。

```aria
sp_border id, color, width
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `color` | 文字列 | 枠線色（`#rrggbb`） |
| `width` | 整数 | 枠線の太さ（ピクセル） |

**例:**
```aria
sp_border 100, "#ffffff", 2     ; 白の2px枠線
```

---

### `sp_gradient`

矩形スプライトのグラデーションを設定します。

```aria
sp_gradient id, _, toColor, direction
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `_` | 文字列 | 未使用（`sp_fill`の開始色が自動的に使用されます） |
| `toColor` | 文字列 | グラデーション終了色（`#rrggbb`） |
| `direction` | 文字列 | 方向。`"vertical"`または`"horizontal"` |

**例:**
```aria
sp_fill 100, "#0000ff", 255
sp_gradient 100, "", "#ff0000", "vertical"    ; 青から赤への縦グラデーション
```

**注意:** 開始色は`sp_fill`で設定した`FillColor`が使用されます。`direction`のデフォルトは`"vertical"`です。

---

### `sp_shadow`

スプライトにドロップシャドウを追加します。

```aria
sp_shadow id, offsetX, offsetY, color, alpha
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `offsetX` | 整数 | 影のX方向オフセット |
| `offsetY` | 整数 | 影のY方向オフセット |
| `color` | 文字列 | 影の色（`#rrggbb`） |
| `alpha` | 整数 | 影の不透明度（0〜255） |

**例:**
```aria
sp_shadow 100, 4, 4, "#000000", 128     ; 右下へ4pxの影
```

## アニメーション

### `ease`

以降のアニメーションコマンド（`amsp`, `afade`, `ascale`）で使用するイージング関数を設定します。

```aria
ease type
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `type` | 文字列 | `"linear"` / `"easein"` / `"easeout"` / `"easeinout"` |

**例:**
```aria
ease "easeout"
amsp 1, 500, 200, 500       ; easeoutで移動
```

**注意:** デフォルトは`"linear"`です。`ease`の設定はグローバルで、次に`ease`を呼ぶまで維持されます。

---

### `amsp`

スプライトをアニメーション付きで移動します。

```aria
amsp id, toX, toY, duration
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `toX` | 整数 | 移動先X座標 |
| `toY` | 整数 | 移動先Y座標 |
| `duration` | 整数 | アニメーション時間（ミリ秒） |

**例:**
```aria
ease "easeout"
amsp 1, 500, 200, 500       ; 500msかけて(500, 200)へ移動
```

---

### `afade`

スプライトをアニメーション付きでフェードします。

```aria
afade id, toAlpha, duration
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `toAlpha` | 整数 | 目標不透明度（0〜255） |
| `duration` | 整数 | アニメーション時間（ミリ秒） |

**例:**
```aria
afade 1, 0, 1000            ; 1000msかけて透明に
```

---

### `ascale`

スプライトをアニメーション付きで拡大・縮小します。

```aria
ascale id, toScaleX, toScaleY, duration
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `toScaleX` | 数値 | 目標X拡大率 |
| `toScaleY` | 数値 | 目標Y拡大率 |
| `duration` | 整数 | アニメーション時間（ミリ秒） |

**例:**
```aria
ascale 1, 1.5, 1.5, 300     ; 300msかけて1.5倍に拡大
```

---

### `acolor`

スプライトの色を即座に変更します。

```aria
acolor id, color
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `id` | 整数 | スプライトID |
| `color` | 文字列 | 新しい色（`#rrggbb`） |

**例:**
```aria
acolor 10, "#ff0000"        ; テキストを即座に赤に
```

**注意:** `acolor`はアニメーション**ではなく**即座に色を変更します。名前に`a`がついていますが、トゥイーン対象外です。

---

### `await`

アニメーションの完了を待ちます。

```aria
await
```

**例:**
```aria
amsp 1, 500, 200, 500
await                       ; 移動が終わるまで待機
text "移動完了しました"
```

**注意:** `await`を呼ぶと、VMは`WaitingForAnimation`状態になり、すべてのトゥイーンが終了すると自動的に復帰します。タイプライター効果が完了していない場合は、そちらも待ちます。

## 画面エフェクト

### `fade_in`

画面をフェードインします。

```aria
fade_in [duration]
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `duration` | 整数 | フェード時間（ミリ秒）。省略時は1000ms |

**例:**
```aria
fade_in         ; 1000msでフェードイン
fade_in 500     ; 500msでフェードイン
```

---

### `fade_out`

画面をフェードアウトします。

```aria
fade_out [duration]
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `duration` | 整数 | フェード時間（ミリ秒）。省略時は1000ms |

**例:**
```aria
fade_out        ; 1000msでフェードアウト
fade_out 2000   ; 2秒かけてフェードアウト
```

---

### `quake`

画面を揺らします。

```aria
quake [amplitude] [time]
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `amplitude` | 整数 | 揺れの振幅（ピクセル）。省略時は5 |
| `time` | 整数 | 揺れ時間（ミリ秒）。省略時は500 |

**例:**
```aria
quake           ; デフォルト（振幅5、500ms）
quake 10, 1000  ; 振幅10、1秒間揺れる
```

**注意:** 揺れは現在のカメラ/描画オフセットに適用されます。すべてのスプライトが一緒に揺れます。

## 背景

### `bg`

背景画像または背景色を設定します。

```aria
bg "path"
bg "#color"
```

| 引数 | 型 | 説明 |
|------|-----|------|
| `path` | 文字列 | 背景画像パス、または`#rrggbb`形式の色コード |

**例:**
```aria
bg "bg/forest.png"          ; 画像を背景に
bg "#000000"                ; 黒背景
bg "#1a1a2e"                ; カスタム色背景
```

**注意:**
- 背景はスプライトID `0`に設定されます
- 色コードを指定した場合、画面サイズの矩形スプライトが作成されます
- `load_bg`は`bg`と完全に同じ機能です（互換性用の別名）

| `ascale` | ✓ | ✓ | ✓ | アニメ拡大 |
| `acolor` | - | ✓ | ✓ | 色変更（即座） |

## 総合例

```aria
*start
    ; 背景設定
    bg "bg/title.png"

    ; キャラクター表示
    lsp 1, "ch/mio_normal.png", 300, 100
    sp_z 1, 10

    ; UIパネル（角丸矩形）
    lsp_rect 100, 50, 400, 300, 80
    sp_fill 100, "#2a2a3e", 230
    sp_round 100, 16
    sp_border 100, "#4a4a6e", 2
    sp_shadow 100, 4, 4, "#000000", 128

    ; テキストラベル
    lsp_text 101, "スタート", 100, 425
    sp_color 101, "#ffffff"
    sp_fontsize 101, 24
    sp_text_align 101, "center"
    sp_text_shadow 101, 2, 2, "#000000"

    ; ボタン化（spbtnはInputカテゴリ）
    spbtn 100, 1
    btnwait %result

    if %result == 1
        ; フェードアウトしてシーン変更
        fade_out 500
        await
        csp -1
        bg "bg/stage1.png"
        fade_in 500
    endif
```
