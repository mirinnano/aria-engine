## 待機・タイマー

### `wait` / `delay`

指定時間だけスクリプト実行を待機します。引数なしで呼び出すと、クリック待機になります。

**カテゴリ**: Core  
**エイリアス**: `delay` = `wait`

```aria
wait [ミリ秒]
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| ミリ秒 | 整数 | 省略可 | 待機時間（ミリ秒）。省略時はクリック待機 |

**例**:
```aria
wait 1000          ; 1秒待機
wait               ; クリック待機（テキスト送り）
delay 500          ; wait と同じ
```

**注意**:
- 引数なしの `wait` は `State.State = WaitingForClick` に設定し、オートセーブも実行します
- スキップモード中は待機をスキップできます

---

### `resettimer`

スクリプトタイマーを0にリセットします。

**カテゴリ**: Core

```aria
resettimer
```

**例**:
```aria
resettimer
; ... 処理 ...
gettimer %elapsed
```

---

### `gettimer`

スクリプト開始（または最後の `resettimer`）からの経過時間（ミリ秒）をレジスタに取得します。

**カテゴリ**: Core

```aria
gettimer %dest
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| %dest | 整数レジスタ | ○ | 経過時間を格納するレジスタ |

**例**:
```aria
resettimer
wait 500
gettimer %t        ; %t ≈ 500
```

---

### `waittimer`

スクリプトタイマーが指定値に達するまで待機します。

**カテゴリ**: Core

```aria
waittimer %dest, 目標値
waittimer 目標値    ; 最小引数1個
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| 目標値 | 整数 | ○ | 待機するタイマー値（ミリ秒） |

**例**:
```aria
resettimer
waittimer 2000     ; タイマーが2000msに達するまで待機
```

**注意**:
- 内部的には `DelayTimerMs = max(0, target - ScriptTimerMs)` を設定します
- 既にタイマーが目標値を超えている場合は即座に復帰します

## 変数代入

### `mov` / `let`

整数レジスタまたは文字列レジスタに値を代入します。

**カテゴリ**: Core  
**エイリアス**: `let` = `mov`

```aria
mov %dest, 値
let %dest, 値
mov $dest, "文字列"
let $dest, "文字列"
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| %dest / $dest | レジスタ | ○ | 代入先レジスタ |
| 値 | 整数/文字列 | ○ | 代入する値、または別のレジスタ |

**例**:
```aria
mov %0, 100
let %health, %maxhealth
mov $name, "ミオ"
let %result, %a + %b     ; 式評価も対応
```

**注意**:
- `let` は `=` を含む構文（`let %x = 100`）も許容し、自動的に除去します
- 配列アクセス構文 `let %arr[0], 10` は `setarray` と同等に動作します
- 文字列補間 `${$name}` や式 `${%a + %b}` に対応しています
- ローカルスコープ内ではローカル変数に優先して書き込みます
- `global` スコープ指定でグローバル変数への直接書き込みが可能です

## 算術演算

### `add`

レジスタに値を加算します。

**カテゴリ**: Core

```aria
add %dest, 値
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| %dest | 整数レジスタ | ○ | 加算先レジスタ（結果もここに格納） |
| 値 | 整数/レジスタ | ○ | 加算する値 |

**例**:
```aria
mov %score, 0
add %score, 100    ; %score = 100
add %score, %bonus ; %score = 100 + bonus
```

---

### `sub`

レジスタから値を減算します。

**カテゴリ**: Core

```aria
sub %dest, 値
```

**例**:
```aria
mov %hp, 100
sub %hp, 20        ; %hp = 80
```

---

### `mul`

レジスタに値を乗算します。

**カテゴリ**: Core

```aria
mul %dest, 値
```

**例**:
```aria

```

---

### `div`

レジスタを値で除算します。

**カテゴリ**: Core

```aria

```

**例**:
```aria

```

**注意**:
- 0による除算はエラーを報告し、結果を0として扱います

**注意**:
- 0による剰余演算はエラーを報告し、結果を0として扱います

## 比較・分岐

### `cmp`

2つの値を比較し、結果を内部比較フラグに設定します。

**カテゴリ**: Core

```aria
cmp %a, %b
cmp %a, 10
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| 左辺 | 整数レジスタ | ○ | 比較対象A（現在のレジスタ値） |
| 右辺 | 整数/レジスタ | ○ | 比較対象B |

**結果**:
- 等しい → `CompareFlag = 0`
- 左辺 > 右辺 → `CompareFlag = 1`
- 左辺 < 右辺 → `CompareFlag = -1`

**例**:
```aria
cmp %0, 1
beq *label_equal
```

---

### `beq`

比較フラグが「等しい」（0）の場合、指定ラベルにジャンプします。

**カテゴリ**: Core

```aria
beq *label
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| *label | ラベル名 | ○ | ジャンプ先ラベル |

**例**:
```aria
cmp %choice, 1
beq *route_a
cmp %choice, 2
beq *route_b
```

---

### `bne`

比較フラグが「等しくない」（非0）の場合、指定ラベルにジャンプします。

**カテゴリ**: Core

```aria
bne *label
```

**例**:
```aria
cmp %flag, 0
bne *already_seen
```

---

### `bgt`

比較フラグが「大きい」（1）の場合、指定ラベルにジャンプします。

**カテゴリ**: Core

```aria
bgt *label
```

---

### `blt`

比較フラグが「小さい」（-1）の場合、指定ラベルにジャンプします。

**カテゴリ**: Core

```aria
blt *label
```

---

### `jmp` / `goto`

無条件に指定ラベルにジャンプします。

**カテゴリ**: Core  
**エイリアス**: `goto` = `jmp`

```aria
jmp *label
goto *label
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| *label | ラベル名 | ○ | ジャンプ先ラベル |

**例**:
```aria
goto *main_menu
jmp *chapter_2
```

**注意**:
- ジャンプ前に現在のスコープ内の `defer` がすべて実行されます
- スコープスタックはリセットされます

## ループ制御

### `for` / `next`

指定回数ループします。

**カテゴリ**: Core

```aria
for %var = 開始値 to 終了値
    ; 繰り返し処理
next
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| %var | 整数レジスタ | ○ | ループ変数 |
| 開始値 | 整数 | ○ | 初期値 |
| 終了値 | 整数/配列名 | ○ | 終了値（含む）。配列名を指定すると配列長-1でループ |

**例**:
```aria
for %i = 0 to 4
    text "ループ回数: %i"
    wait 200
next
```

**注意**:
- `next` はループ変数を1増やし、終了値を超えていなければ `for` の次の行に戻ります
- 配列名を終了値に指定すると、配列の要素数でループします

---

### `while` / `wend`

条件が真である間、ブロックを繰り返します。

**カテゴリ**: Core

```aria
while 条件
    ; 繰り返し処理
wend
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| 条件 | 式 | ○ | 継続条件（真の間ループ） |

**例**:
```aria
mov %count, 0
while %count < 5
    text "カウント: %count"
    inc %count
wend
```

**注意**:
- `while`/`wend` はパーサーによって `JumpIfFalse` と `jmp` に展開されます
- ブロック形式 `while cond { ... }` もサポートされています

---

### `break`

現在のループを中断します。

**カテゴリ**: Core

```aria
break
```

**例**:
```aria
while %running == 1
    gettimer %t
    if %t > 5000
        break
    endif
wend
```

**注意**:
- パーサーによってループ終端ラベルへの `jmp` に展開されます
- スコープ内にいる場合、スコープを抜ける際の `defer` が実行されます

---

### `continue`

現在のループの残りをスキップし、次の繰り返しに進みます。

**カテゴリ**: Core

```aria
continue
```

**例**:
```aria
for %i = 0 to 9
    if %i == 3
        continue
    endif
    text "%i"
next
```

**注意**:
- パーサーによってループ先頭ラベルへの `jmp` に展開されます
- スコープ内にいる場合、スコープを抜ける際の `defer` が実行されます

## 配列操作

### `setarray`

整数配列の指定インデックスに値を設定します。配列が存在しない場合は自動作成されます。

**カテゴリ**: Core

```aria
setarray %array名, インデックス, 値
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| %array名 | 配列名 | ○ | 配列の名前 |
| インデックス | 整数 | ○ | 設定する位置（0始まり） |
| 値 | 整数 | ○ | 設定する値 |

**例**:
```aria
setarray %scores, 0, 100
setarray %scores, 1, 85
setarray %scores, 2, 92
```

**注意**:
- 負のインデックスは無視されます
- インデックスが配列サイズを超える場合、自動的に拡張されます

---

### `getarray`

整数配列の指定インデックスの値をレジスタに取得します。

**カテゴリ**: Core

```aria
getarray %dest, %array名, インデックス
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| %dest | 整数レジスタ | ○ | 値を格納するレジスタ |
| %array名 | 配列名 | ○ | 配列の名前 |
| インデックス | 整数 | ○ | 取得する位置（0始まり） |

**例**:
```aria
getarray %score, %scores, 0    ; %score = 100
```

**注意**:
- 範囲外のインデックスの場合、0が設定されます


| パラメータ | 型 | 必須 | 説明 |

## 例外・デバッグ

### `throw`

例外をスローします。`try` スタックにキャッチ位置が積まれている場合はそこにジャンプし、ない場合はVMを終了します。

**カテゴリ**: Core

```aria
throw
```

**例**:
```aria
if %error == 1
    throw
endif
```

**注意**:
- スコープを抜ける際の `defer` が実行されます

---

### `assert`

条件が偽の場合にエラーを報告します（開発時のデバッグ用）。

**カテゴリ**: Core

```aria
assert 条件, "メッセージ"
assert 条件
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| 条件 | 式 | ○ | 真であるべき条件 |
| メッセージ | 文字列 | 省略可 | 失敗時のメッセージ |

**例**:
```aria
assert %hp >= 0, "HPが負数になりました"
assert %score <= 100
```

---

### `panic`

即座にVMを停止し、エラーを報告します。

**カテゴリ**: Core

```aria
panic "メッセージ"
panic
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| メッセージ | 文字列 | 省略可 | 停止理由のメッセージ |

**例**:
```aria
if %critical_error == 1
    panic "クリティカルエラーが発生しました"
endif
```

## 遅延実行

### `defer`

現在のスコープを抜ける際に実行する命令を登録します。LIFO（後登録先実行）順で実行されます。

**カテゴリ**: Core

```aria
defer コマンド名, [引数...]
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| コマンド名 | 文字列 | ○ | 遅延実行するコマンド名 |
| 引数... | 任意 | 省略可 | コマンドの引数（レジスタは定義時に評価） |

**例**:
```aria
scope
    defer csp, 100
    lsp 100, "temp.png", 0, 0
    ; ... 処理 ...
end_scope          ; ここで csp 100 が自動実行される
```

**注意**:
- 引数に `%` レジスタを指定した場合、**定義時の値**が固定されます
- 引数に `$` レジスタを指定した場合、**定義時の文字列値**が固定されます
- スコープ外で使用した場合は何も起こりません

## テキスト表示

### `text`

テキストを表示します。パーサーによる特殊処理が行われ、CommandRegistryには登録されていません。

**カテゴリ**: 特殊（パーサー処理）

```aria
text "表示する文字列"
キャラ名「セリフ」
```

| パラメータ | 型 | 必須 | 説明 |
|-----------|-----|------|------|
| 文字列 | 文字列 | ○ | 表示するテキスト |

**例**:
```aria
text "ようこそ、AriaEngineへ！"
ミオ「こんにちは！"
```

**動作詳細**:
- テキストを現在のテキストバッファに追加します
- `text_target` で指定したスプライト（または互換モード時の自動生成スプライト）に描画されます
- `textspeed` が設定されている場合、タイプライター効果で1文字ずつ表示されます
- 文字列補間 `${$name}` や式 `${%0 + 1}` に対応しています
- セリフ形式（`名前「セリフ」`）を使用すると、自動的に `textclear` とページ送り（`\`）が挿入されます

**注意**:
- `compat_mode off` の場合、`text_target` の設定が必要です
- 互換モードでは自動的にテキストボックス背景とテキストスプライトが生成されます
