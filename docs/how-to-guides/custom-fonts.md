# カスタムフォントの使い方

このガイドでは、AriaEngine で独自のフォントを使う方法を説明します。ゲームの雰囲気に合ったフォントを選ぶことで、世界観の統一感を高められます。

---

## 対応フォーマット

AriaEngine は **TTF（TrueType Font）** 形式のフォントに対応しています。

- `.ttf` ファイルを用意してください
- `.otf` などのその他のフォント形式は読み込めません
- 日本語表示には、日本語グリフを含んだフォントファイルが必要です

推奨フォントの例:

- [Noto Sans JP](https://fonts.google.com/noto/specimen/Noto+Sans+JP)（Google Fonts、無料）
- [M PLUS 1p](https://fonts.google.com/specimen/M+PLUS+1p)（Google Fonts、無料）
- [源ノ角ゴシック](https://github.com/adobe-fonts/source-han-sans)（Adobe、無料）

---

## フォントファイルの配置

プロジェクト内の任意のフォルダに `.ttf` ファイルを置けます。一般的には `assets/fonts/` フォルダを作って管理します。

```
engine/
├── src/AriaEngine/
└── assets/
    └── fonts/
        └── NotoSansJP-Regular.ttf
```

フォントファイル名は半角英数字と記号を使うのが確実です。日本語ファイル名でも動作しますが、環境によっては問題が出る可能性があるので避けてください。

---

## init.aria でフォントを設定する

`init.aria` に3つのコマンドを追加します。これらは **init-only**（エンジン起動時に一度だけ読み込まれる）です。

```aria
; フォントファイルのパス
font "assets/fonts/NotoSansJP-Regular.ttf"

; フォントアトラスのサイズ
font_atlas_size 256

; テクスチャフィルタモード
font_filter "bilinear"
```

### font コマンド

使用するフォントファイルを指定します。

```aria
font "<パス>"
```

- `init.aria` 専用です。メインスクリプトで変更しても反映されません
- パスはエンジンの実行ディレクトリからの相対パスです
- 未設定の場合、フォントが読み込まれず警告が出ます

### font_atlas_size コマンド

フォントテクスチャアトラスのサイズを設定します。

```aria
font_atlas_size <サイズ>
```

| 設定値 | 用途 |
|--------|------|
| `128` | 英語のみの短いスクリプト |
| `256` | 日本語あり、通常の長さ（デフォルト） |
| `512` | 日本語あり、長いスクリプトや多くの文字種 |

- 値は `8` から `512` の範囲に自動的に調整されます
- 大きくすると文字がよりきれいになりますが、GPU メモリを多く使います
- 多くの文字を使うゲームでは大きめの値を選んでください

### font_filter コマンド

フォントの表示品質を調整します。

```aria
font_filter "<モード>"
```

| モード | 効果 | 用途 |
|--------|------|------|
| `bilinear` | バイリニア補間（標準） | ほとんどのゲームに最適 |
| `trilinear` | トリリニア補間 | 高品質表示が必要な場合 |
| `anisotropic` | 異方性フィルタリング | `trilinear` と同じ設定にフォールバック |
| `point` | ニアレストネイバー | ドット絵風やピクセルアート風の演出 |

- デフォルトは `bilinear` です
- `point` にすると文字の輪郭がギザギザになり、レトロな雰囲気が出せます
- `trilinear` / `anisotropic` は小さい文字でも読みやすくなります

---

## 完全な設定例

```aria
; ウィンドウ設定
window 1280, 720, "フォント設定例"

; フォント設定
font "assets/fonts/NotoSansJP-Regular.ttf"
font_atlas_size 256
font_filter "bilinear"

; メインスクリプト
script "assets/scripts/main.aria"

; テキストボックス設定（ランタイムでも変更可能）
textbox 50, 500, 1180, 200
fontsize 32
textcolor "#ffffff"
```

## フォント設定の確認

フォントが正しく読み込まれたかは、エンジン起動時のコンソール出力で確認できます。

読み込み成功時:
```
(Font loaded successfully)
```

読み込み失敗時:
```
RENDER_FONT_LOAD: フォント 'assets/fonts/xxx.ttf' を読み込めませんでした。
```

失敗した場合は、パスが正しいか、ファイルが破損していないかを確認してください。

## トラブルシューティング

### 文字が表示されない、四角や空白になる

- フォントファイルに日本語グリフが含まれているか確認してください
- 英語のみのフォントでは日本語が表示できません
- `font_atlas_size` を大きくしてみてください（256 や 512）

### フォントがぼやけて見える

- `font_filter` を `trilinear` に変更してみてください
- `font_atlas_size` を大きくするとアンチエイリアス効果が向上します

### フォント読み込みで警告が出る

```
RENDER_FONT_LOAD: フォント '...' を読み込めませんでした。
```

- パスが正しいか確認してください（相対パスはエンジンの実行ディレクトリ基準）
- ファイルが存在するか確認してください
- TTF 形式か確認してください（OTF は非対応）
- フォントファイルが破損していないか確認してください

### 特定の文字だけ表示されない

- エンジンはスクリプトファイルに含まれる文字を自動的に収集してアトラスを生成します
- スクリプトにその文字が含まれていないと、フォントアトラスに組み込まれません
- 表示したい文字がスクリプト内に含まれているか確認してください
